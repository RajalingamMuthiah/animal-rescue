## ğŸ¬ Interaction States

### Button Hover State
```

  Position: Y = 0
  Shadow: 0 8px 25px

On Hover:
  Position: Y = -3px (lifted)
  Shadow: 0 12px 35px (enhanced)
  Background: Darker or filled
  Transition: 0.3s ease
```

### Button Active State
```
On Click/Press:
  Position: Y = -1px (slightly lifted)
  Duration: Immediate
  Transition: 0.1s
```
